# Vanish-FFConverter

Простая графическая оболочка для **ffmpeg** под Windows. Цель — сделать
максимально понятный инструмент для повседневных задач с аудио и видео,
чтобы не держать в голове командную строку ffmpeg. Одна портативная
программа + папка `ffmpeg`, без установки в систему.

> ⚠️ **Проект в разработке.** Сейчас доделывается первая вкладка —
> **«Аудио»**. Остальные вкладки присутствуют в интерфейсе как заглушки
> и будут реализованы в следующих версиях.

![Вкладка «Аудио»](assets/audio-tab.jpg)

## Требования

- Windows 10/11, 64-bit.
- .NET Framework 4.8 — уже встроен в Windows 10/11, ставить не нужно.
- Интернет один раз — чтобы скрипт скачал ffmpeg (~180 МБ).

## Установка и запуск

### Вариант 1. Просто пользоваться (готовая сборка)

1. Откройте раздел **Releases** (справа на странице проекта) и скачайте
   архив `Vanish-FFConverter-…-win64.zip`.
2. **Распакуйте** его в любую папку.
3. **Двойной клик по `get-ffmpeg.bat`** — скрипт скачает ffmpeg в
   подпапку `ffmpeg\`. Нужен один раз; требуется интернет. По окончании
   он напишет «Готово».
4. **Двойной клик по `Vanish-FFConverter.exe`** — программа откроется.

### Вариант 2. Собрать самому (из исходников)

1. **Скачайте код** — зелёная кнопка «Code» → «Download ZIP» (или
   `git clone`) — и распакуйте в любую папку.
2. **Двойной клик по `build.bat`** — он при необходимости сам скачает
   ffmpeg, затем соберёт `Vanish-FFConverter.exe` встроенным в Windows
   компилятором `csc` (никаких SDK ставить не нужно).
3. **Двойной клик по `Vanish-FFConverter.exe`**.

При падениях рядом с программой создаётся `crash.log`.

## Перенос на другой компьютер

Скопируйте всю папку целиком — и всё, программа портативная: настройки
(`settings.json`) и ffmpeg едут внутри, в систему ничего не ставится.
Требование к машине — только Windows 10/11 64-bit. **Пробелы и кириллица
в пути не мешают** (проверено, в т.ч. в скрипте загрузки ffmpeg).

## Почему такой стек

Написано на **C# + WinForms** (.NET Framework 4.8). Такой выбор — ради
трёх вещей:

- **Минимум зависимостей.** .NET Framework 4.8 уже встроен в Windows
  10/11 — ставить ничего не нужно. Никакого NuGet, никаких сторонних
  библиотек.
- **Минимальный размер.** Сам исполняемый файл — сотни килобайт. Весь
  «вес» — это папка ffmpeg, которую качает скрипт.
- **Портативность.** Одна папка, без установщика, без записи в реестр.

Сборка идёт встроенным компилятором `csc.exe` из состава .NET Framework —
без внешних SDK. Программа сама медиа не декодирует — она вызывает
`ffmpeg`/`ffprobe`/`ffplay` как внешние процессы.

## Вкладка «Аудио» — что уже умеет

- **Форматы:** opus, mp3, aac (в контейнере .m4a), flac, wav.
- **Пресеты качества** (компактный/стандарт/высокое/максимум) и
  **расширенный режим** с ручной настройкой битрейта, режима CBR/VBR,
  частоты дискретизации и полем «доп. параметры ffmpeg» (с встроенной
  справкой по фильтрам).
- **Нормализация громкости** по стандарту EBU R128 (двухпроходный
  loudnorm) с выбором целевого уровня LUFS.
- **Работа с дорожками мультитрека:** выбор нужных дорожек, сведение в
  моно, «голоса по ушам» (стерео), либо каждая дорожка в свой файл.
- **Очередь заданий:** постановка отдельных файлов или всех сразу,
  запуск/стоп, пауза после текущего, пропуск, «запустить только это
  задание» (правый клик по файлу).
- **Прогресс и логи:** зелёный прогресс-бар по текущему файлу, выезжающая
  справа консоль с полной командой ffmpeg, полный лог-файл и краткий итог
  по каждому циклу.
- **Удобства:** перетаскивание файлов в окно, шаблоны имён результата,
  вывод рядом с исходником или в выбранную папку, тёмная и светлая тема.

## Что где лежит

| Папка/файл | Что это |
|---|---|
| `Vanish-FFConverter.exe` | сама программа (иконка — `V.ico`) |
| `get-ffmpeg.bat` / `.ps1` | скачивают ffmpeg в папку `ffmpeg\` |
| `build.bat` | сборка из исходников встроенным `csc` |
| `src\` | исходный код (C#) |
| `ffmpeg\` | ffmpeg + ffprobe + ffplay (качается скриптом) |
| `logs\` | лог-файлы обработок и краткие итоги |
| `temp\` | временные файлы (чистится само) |
| `settings.json` | настройки программы (запоминаются при закрытии) |
| `docs\` | техзадание проекта |

## О ffmpeg

ffmpeg не входит в репозиторий и скачивается отдельно скриптом
`get-ffmpeg.bat` (сборка [BtbN FFmpeg-Builds](https://github.com/BtbN/FFmpeg-Builds),
win64, GPL). ffmpeg распространяется под собственной лицензией.

## Принципы проекта

Главный принцип — **простота важнее полноты**. Программа сознательно
остаётся лёгкой обёрткой над ffmpeg и **не планирует** обрастать тяжёлым
кодом: встроенного предпросмотра/плеера файлов, таймлайна, визуальной
покадровой резки здесь не будет — это утяжеляет программу и дублирует
специализированные редакторы. Для визуальной резки «глазами» удобнее,
например, Machete. Задача Vanish-FFConverter — быстро и понятно выполнять
основные команды ffmpeg, оставаясь маленькой и портативной.
